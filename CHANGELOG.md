# Changelog

All notable changes to AutoBug will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added - Initial Release
- Complete reconnaissance pipeline with Subfinder, httpx, Naabu integration
- Vulnerability scanning with Nuclei (5000+ templates)
- Intelligent change detection ("Diff Engine")
- Multi-platform scope monitoring (HackerOne, Bugcrowd)
- Multi-channel alerting (Discord, Slack)
- FastAPI REST API with 20+ endpoints
- Modern web dashboard with HTMX
- JWT-based authentication system
- Real-time WebSocket updates
- Export functionality (CSV/JSON)
- PostgreSQL database with SQLAlchemy ORM
- Redis caching and job queuing
- Prefect workflow orchestration
- Docker Compose deployment
- Comprehensive CLI interface
- Rate limiting and security middleware
- Role-based access control
- Daily/weekly digest reports
- Asset scope validation
- Historical tracking for programs and scans
